保存履歴

このWeb版は端末内ブラウザに保存します。別端末へ移す場合はCSV出力を使ってください。

PDF用レポート

かんたん使い方ガイド

STEP1 受講者名を入力して「ID取得」を押します。

STEP2 分かる範囲で基本情報・アワード数を入力します。全部埋めなくても保存できます。

STEP3 「AI総評」を押すと入力内容から振り返り文を作成します。

STEP4 「保存」で端末内に記録します。

STEP5 「PDF」でレポートを表示し、スマホの共有/印刷機能でPDF保存できます。
